# Amazon Elastic Block Store (Amazon EBS)

¿Qué es?

**Amazon Elastic Block Store (Amazon EBS)** es un servicio de almacenamiento en bloque de alto rendimiento y durabilidad para utilizar con instancias de **Amazon EC2** (Elastic Compute Cloud). EBS ofrece almacenamiento persistente que se puede utilizar para ejecutar sistemas de archivos, bases de datos y aplicaciones que requieren acceso rápido a discos y necesitan almacenamiento duradero.
