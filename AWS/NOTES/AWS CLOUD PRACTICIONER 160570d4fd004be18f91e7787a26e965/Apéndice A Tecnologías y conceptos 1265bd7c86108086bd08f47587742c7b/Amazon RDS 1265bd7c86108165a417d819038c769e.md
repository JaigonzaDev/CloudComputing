# Amazon RDS

¿Qué es?

**Amazon RDS (Relational Database Service)** es un servicio de base de datos en la nube completamente administrado que facilita la configuración, el funcionamiento y la escalabilidad de bases de datos relacionales en la nube. Ofrece una forma simple y eficiente de administrar bases de datos tradicionales como MySQL, PostgreSQL, MariaDB, Oracle, y Microsoft SQL Server, sin tener que preocuparte por la gestión del hardware y la infraestructura subyacente.

¿Como funciona?

Básicamente contiene todas las caracteristicas de ser administrado y lo hace compatible con los servicios de bases de datos tradicionales.
