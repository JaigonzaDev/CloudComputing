# AWS Storage Gateway

¿Qué es?

**AWS Storage Gateway** es un servicio que conecta aplicaciones locales con el almacenamiento en la nube de AWS. Actúa como un puente entre tu infraestructura local y la nube, permitiendo a las empresas usar el almacenamiento en la nube de manera eficiente y transparente, sin necesidad de cambios significativos en sus aplicaciones.

¿Como funciona?

Funciona instalando un software en tu servidor local, que actúa como un **puente** entre tu infraestructura local y el almacenamiento en la nube de AWS
