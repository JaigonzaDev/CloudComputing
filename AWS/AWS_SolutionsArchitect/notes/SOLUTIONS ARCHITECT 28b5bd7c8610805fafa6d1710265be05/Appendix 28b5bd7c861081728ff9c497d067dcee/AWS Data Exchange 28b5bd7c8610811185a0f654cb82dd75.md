# AWS Data Exchange

https://docs.aws.amazon.com/es_es/data-exchange/latest/userguide/what-is.html

¿Qué es?

AWS Data Exchange es un servicio que ayuda a AWS los clientes a compartir y gestionar fácilmente los derechos de datos de otras organizaciones a gran escala.

Como receptor de datos, puede realizar un seguimiento y gestionar todas sus concesiones de datos y suscripciones de AWS Marketplace datos en un solo lugar. Cuando tiene acceso a un conjunto de AWS Data Exchange datos, puede utilizar análisis y aprendizaje automático compatibles AWS o asociados para extraer información de él.
